\chapter{Entorno computacional}\label{apendice:entorno}

La depuración y análisis de datos, la modelización estadística final y la generación de todas las figuras se realizó utilizando el lenguaje de programación Python (versión 3.12). Para garantizar la reproducibilidad, el análisis se llevó a cabo en un entorno virtual aislado.

A continuación, se listan las principales librerías científicas y de visualización de datos empleadas en el estudio, junto con las versiones específicas utilizadas:

\begin{itemize}
    \item Pandas (versión 2.3.1): Para la manipulación y limpieza de datos tabulares.
    \item NumPy (versión 1.26.0): Para operaciones numéricas y algebraicas.
    \item SciPy (versión 1.16.0): Utilizada para el ajuste de distribuciones de probabilidad en el cálculo del SPEI.
    \item Linearmodels (versión 6.1): Para la estimación de los modelos de panel de efectos fijos y aleatorios.
    \item Statsmodels (versión 0.14.5): Utilizada para pruebas estadísticas complementarias.
    \item Scikit-learn (versión 1.7.1): Para la implementación del algoritmo de agrupamiento K-Means.
    \item GeoPandas (versión 1.1.1): Para la manipulación de datos geoespaciales (shapefiles).
    \item Plotly (versión 6.2.0): Para la creación de visualizaciones interactivas, incluyendo los mapas.
    \item Matplotlib (versión 3.10.3): Para la generación de gráficos estáticos.
    \item Seaborn (versión 0.13.2): Para la creación de visualizaciones estadísticas.
    \item Tabulate (versión 0.9.0): Para la exportación de tablas a formato LaTeX.
    \item SPEI (versión 0.8.0) y Climate-Indices (versión 1.0.12): Librerías utilizadas y evaluadas para el cálculo de índices de sequía.
\end{itemize}

El conjunto de datos y el código fuente utilizado para el análisis se encuentra disponible para su consulta, garantizando la total transparencia y replicabilidad de los resultados de esta tesis, en un repositorio de GitHub de acceso abierto, cuyo enlace se proporciona:
\url{https://github.com/hvargasojeda/tesis_sequia_2.git}

\chapter{Fuente oficial de los datos}\label{apendice:fuente_datos}

Los datos fueron solicitados a la Dirección de Meteorología e Hidrología (DMH) del Paraguay, a través de un pedido formal por intermedio de la Facultad de Ciencias Exactas y Naturales de la Universidad Nacional de Asunción, vía correo electrónico. La solicitud se realizó el 24 de abril de 2024, y se obtuvo una respuesta oficial el 9 de mayo de 2024, proporcionando los datos en formato digital (xlsx).
